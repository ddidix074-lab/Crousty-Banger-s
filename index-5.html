Publication

Mettez votre site en ligne pour le monde entier, sans dépendre de personne.

📌 Comment ça marche : vos modifications sont enregistrées en brouillon sur cet appareil. Pour les rendre visibles par tous vos clients, cliquez sur « Publier » : un fichier index.html complet est téléchargé. Remplacez l'ancien fichier par celui-ci sur votre hébergement (GitHub Pages, OVH…), et le site est à jour partout.

1. Publier le site en ligne

Génère le fichier final du site, contenu intégré, prêt à mettre en ligne. C'est ce fichier que voient vos visiteurs.

2. Sauvegarde de sécurité

Exportez une copie de tout votre contenu (produits, photos, textes, réglages) dans un petit fichier, à garder précieusement. Réimportez-le sur n'importe quel appareil pour reprendre l'édition.

Mettre en ligne sur GitHub Pages (rappel)

  1. Créez un dépôt public sur GitHub.
  2. Uploadez le fichier index.html généré ci-dessus.
  3. Settings → Pages → Branch main / root → Save.
  4. Votre lien : https://votre-pseudo.github.io/nom-du-depot/
  5. À chaque mise à jour : republiez et ré-uploadez le fichier.
